# 设备看板原型设计描述

## 页面结构

页面布局：左侧导航栏 + 右侧内容区

左侧导航
- 固定宽度180px
- 深色背景
- 当前高亮菜单：设备看板

右侧内容区
- 顶部面包屑：设备 / 设备看板
- 顶部右侧：门店名称 + 当前用户

---

## 页面头部区域

左侧：页面标题"设备看板"

右侧从左到右依次：
- 时间筛选栏（仅大项目页签时显示）：今日 / 昨日 / 本周 / 本月 / 自定义，默认选中"今日"；游戏机页签不显示时间选择，只展示当天数据
- 上次刷新时间文字
- 刷新按钮：点击刷新数据，距上次刷新不足1分钟时可点击但无效果

---

## 设备类型页签

一行横向页签：全部 / 收银台 / 自助机 / 游戏机 / 大项目 / 离线设备

- 默认选中"全部"
- "离线设备"页签右上角显示红色圆点标记
- 切换页签时下方内容区联动切换

---

## 全部视图

按顺序纵向排列四个类型：收银台 → 自助机 → 游戏机 → 大项目

每个类型有标题栏，显示类型名称 + 在线数/总数；标题不显示“区块”文字

不显示时间筛选

---

## 收银台

标题栏：收银台 · 3在线/4台 · 当日数据

卡片网格排列，每行4张卡片

在线设备卡片内容：
- 第一行：设备名称（左）+ 在线状态绿点（右）
- 第二行：设备编号（灰色小字）
- 数据区域2列布局：
  - 营收金额：¥12,580
  - 订单数：186
  - 售币数：3,200（仅连接售币机时显示）
  - 剩余实物币：1,800（仅连接售币机时显示）

离线设备卡片内容：
- 设备名称 + 灰色离线状态点
- 设备编号
- 文字提示"设备离线，无营业数据"
- 卡片背景灰色，不可点击

示例数据：
- 1号收银台 C001 在线 营收¥12,580 订单186 售币3,200 剩余实物币1,800（连接售币机）
- 2号收银台 C002 在线 营收¥9,340 订单142 售币2,100 剩余实物币900（连接售币机）
- 3号收银台 C003 在线 营收¥7,820 订单118（未连接售币机，不显示售币和剩余）
- 4号收银台 C004 离线

## 自助机

标题栏：自助机 · 10在线/11台 · 当日数据

每张卡片结构：
- 设备名称 + 状态点
- 设备编号

- 营收金额
- 售币数（含电子币和实物币）
- 剩余实物币数

存币机：
- 存币数量

存票机：
- 存实物票数量

- 营收金额

手持机：
- 营收金额

取卡机：
- 出卡数量
- 剩余卡数量

抽奖机：
- 抽奖总次数

### 按类型视图

纵向排列四个类型分组：

每个分组标题：类型名称 + 类型标签色块
- 模拟机（蓝色标签）
- 彩票机（绿色标签）
- 中性机（橙色标签）
- 礼品机（粉色标签）

每个分组下方为该类型游戏机卡片网格，每行3-4张

游戏机卡片通用结构：
- 第一行：设备名称（左）+ 报警图标🔔红色（右，仅有报警时显示）
- 第二行：设备编号（灰色小字）
- 数据区域（按类型不同）
- 游戏机名称可点击，点击打开P位弹窗

模拟机卡片字段：
- 投币数（蓝色可点击，点击跳转报表）

彩票机卡片字段：
- 投币数
- 出票数
- 出点数
- 每币奖票数（出票数/投币数）

中性机卡片字段：
- 投币数
- 出币数
- 差额（投币数-出币数）
- 盈币率（差额/投币数）

礼品机卡片字段：
- 投币数
- 出礼品数
- 出礼品局数
- 返还率

排序规则：
- 默认按净投币数降序
- 净投币数为0的排在有数据的后面，再按编号升序
- 出彩票排序仅对彩票机生效
- 出礼品排序仅对礼品机生效
- 出点排序对彩票机和中性机均生效

报警图标交互：
- 点击报警图标🔔打开该台游戏机报警明细弹窗（不是点击卡片）
- 点击卡片其它区域打开P位弹窗

## 弹窗设计

### P位弹窗（点击单台游戏机触发）

弹窗标题：设备名称 · P位数据
弹窗宽度：560px
内容：表格

表格列根据游戏机类型不同：

模拟机P位表格列：P位 / 投币数

彩票机P位表格列：P位 / 投币数 / 出票数 / 出点数 / 每币奖票数

中性机P位表格列：P位 / 投币数 / 出点数 / 差额 / 盈币率

礼品机P位表格列：P位 / 投币数 / 出礼品数 / 出礼品局数 / 返还率

## 交互状态汇总

| 交互动作 | 触发位置 | 结果 |
|---|---|---|
| 点击刷新按钮（间隔≥1分钟） | 页面头部 | 刷新所有数据，更新刷新时间 |
| 点击刷新按钮（间隔＜1分钟） | 页面头部 | 按钮可点击但无效果，提示"距上次刷新不足1分钟" |
| 切换设备类型页签 | 页签栏 | 切换下方内容，仅大项目显示时间筛选 |
| 切换时间筛选 | 时间栏 | 刷新大项目数据 |
| 切换游戏机子视图 | 子视图标签 | 切换按类型/按基站/按机种 |
| 切换排序 | 排序栏 | 重新排列游戏机卡片 |
| 点击设备名称 | 收银台、自助机、大项目卡片标题 | 打开该设备对应报表，自动带入当前看板时间和当前设备 |
| 点击游戏机设备名称 | 游戏机卡片标题 | 打开P位弹窗 |
| 点击游戏机报警图标 | 卡片右上角🔔 | 打开该台报警明细弹窗 |
| 点击游戏机指定指标数值 | 游戏机卡片内蓝色下划线数字 | 打开游戏机盈收分析，带当前看板时间+当前设备+点击指标 |
| 点击总看板未处理报警数字 | 总看板 | 打开消息中心，展示游戏机和自助机未处理异常消息 |
| 点击机种名称 | 按机种视图 | 打开机种P位弹窗 |
| 点击普通卡片空白区域 | 所有设备卡片 | 不触发报表或弹窗 |

## 视觉规范

- 在线状态：绿色圆点 + "在线"绿色文字
- 离线状态：灰色圆点 + "离线"灰色文字
- 离线设备卡片：灰色背景#fafafa，不可交互
- 报警图标：红色🔔，hover放大
- 预警标签：橙色底（一般预警）/ 红色底（通讯异常）
- 可点击设备名称与可点击数值：蓝色#0958d9，常态下划线，hover加深
- 总看板：深蓝渐变背景，白色数字
- 弹窗：白色背景，圆角8px，遮罩层半透明黑色
- 卡片hover：不浮动，不改变报表触发范围
- 页签选中：蓝色背景白色文字
- 排序按钮选中：蓝色背景白色文字

## 报表下钻规则

所有报表弹窗统一显示当前看板时间和当前设备。

所有报表查询条件均隐藏合作商。

设备名称下钻规则：

| 设备类型 | 点击位置 | 打开报表 | 额外隐藏条件 |
|---|---|---|---|
| 收银台 | 设备名称 | 销售查询 | 开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端 |
| 售币提币机 | 设备名称 | 销售查询 | 开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端 |
| 售珠提珠机 | 设备名称 | 销售查询 | 开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端 |
| 手持机 | 设备名称 | 销售查询 | 开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端 |
| 存币机 | 设备名称 | 会员存币查询 | 开始时间、结束时间、开始营业日期、结束营业日期、营业班次、设备 |
| 存票机 | 设备名称 | 存实物票查询 | 开始时间、结束时间、设备 |
| 取卡机 | 设备名称 | 不打开报表 | 无 |
| 抽奖机 | 设备名称 | 抽奖机记录 | 开始时间、结束时间 |

## 本轮原型补充

菜单壳：
- 左侧一级菜单按截图一展示。
- 设备看板作为独立二级菜单显示在扣费设备上方。
- 扣费设备分组内不再重复显示设备看板。
- 右侧内容区顶部增加系统标签页。
- 当前打开标签为设备看板。

报表弹窗：
- 查询条件区显示未被隐藏的查询条件。
- 需求指定隐藏的查询条件不显示。
- 查询条件区下方显示查询、导出按钮。
- 不显示“已自动带入当前看板时间和当前设备”的说明条。
- 报表上下文区域只显示看板时间、当前设备，不显示“点击指标”。

游戏机顶部设备看板：
- 投币总数、出票总数、出点总数、出礼品总数均可点击。
- 点击后打开游戏机盈收分析。
- 默认选中按游戏机汇总页签。
- 游戏机顶部看板打开的报表只显示按游戏机汇总页签，其余页签隐藏。
- 隐藏开始营业日期、结束营业日期、合作商。
- 显示游戏机、游戏机分类、营收分类、显示无记录机台。
- 游戏机编号、游戏机名称列固定，横向滚动时不移动。

未处理警报：
- 点击未处理警报数字打开消息中心。
- 消息中心只显示游戏机页签。
- 未处理警报数量只统计游戏机未读异常。
- 点击标记已读后，该消息从列表移除。
- 全部标记已读后，当前页签消息清空。
- 消息列表变化后，未处理警报数字同步更新。

游戏机卡片：
- 单台游戏机卡片显示P位数量，格式为“P位数:2”。
- 机种卡片不显示P位数量。
- 游戏机页签不显示时间选择。
- 游戏机只展示当天数据。
- 游戏机卡片只保留指定指标：模拟机为投币数；彩票机为投币数、出票数、出点数、每币奖票数；礼品机为投币数、出礼品数、出礼品局数、返还率；中性机为投币数、出点数、差额、盈币率。
- 游戏机按机种视图中，礼品机机种卡片不显示出礼品局数。
- 彩票机、礼品机卡片高度固定一致。
- 所有可下钻卡片指标使用蓝色底、边框、下划线作为可点击状态。
- 设备指标名称与数值之间保持清晰间距，避免贴合。
- 除已说明可点击的设备名称、游戏机指定指标和游戏机顶部汇总指标外，其它指标不增加报表弹窗。

顶部栏：
- 搜索栏左侧不显示功能路径“设备/扣费设备/设备看板”。
- 切换系统按钮显示在搜索栏左侧。
- 顶部通栏显示“世软科技”、切换系统、搜索、审批、下载、知识库、更多、版本和用户。
- 一级菜单图标栏从顶部通栏下方开始，图标为方形底图标样式。
- 二级菜单上方不显示“世软科技”和切换系统。

离线设备：
- 离线设备显示最后在线时间和已离线时间。
- 已离线时间小于一个月按天显示。
- 已离线时间超过一个月按月显示。
- 已离线时间超过一年按年显示。
- 删除或失效设备不显示。
