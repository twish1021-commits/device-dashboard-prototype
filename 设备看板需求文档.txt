# 设备看板需求文档

## 一、功能定位与入口

- 定位：设备运营看板，按设备查询对应营业数据
- 位置：设备模块下独立功能菜单
- 菜单名称：设备看板

---

## 二、整体交互规则

**设备加载范围**
- 只加载有效状态设备（含在线、离线）
- 离线超7天的设备不在主列表显示，统一归入"离线设备"页签
- 离线超7天判断基准：设备最后一次上报数据的时间

**默认视图**
- 默认显示全部设备类型
- 显示顺序：收银台 → 自助机 → 游戏机 → 大项目
- 每个类型下有效在线设备优先，同状态内按编号升序排列

**设备类型切换**
- 支持切换显示：全部 / 收银台 / 自助机 / 游戏机 / 大项目
- 切换为游戏机类型后，支持三种子视图切换：按类型 / 按基站 / 按机种
- 游戏机默认子视图：按类型

**时间筛选**
- 仅大项目类型显示时间筛选
- 大项目默认今日，可选：今日 / 昨日 / 本周 / 本月 / 自定义
- 收银台、自助机、游戏机固定显示当日数据，不显示时间筛选
- 全部设备视图不显示时间筛选

**数据刷新**
- 打开功能时自动刷新一次
- 提供刷新按钮，点击后刷新数据
- 距上次刷新不足1分钟时，刷新按钮可点击但不产生效果

**页面滚动**
- 右侧内容区垂直滚动时，页面标题、上次刷新时间、刷新按钮、设备类型页签和看板取数说明按钮固定在内容区顶部
- 下方设备列表独立滚动，不带动上述顶部区域离开可视区

**离线设备页签**
- 独立页签，名称：离线设备
- 页签内按收银台 / 自助机 / 游戏机 / 大项目分类显示
- 每台设备只显示：设备名称 / 设备编号 / 离线状态

---

## 三、收银台

**数据范围：** 当日

**每台显示字段**
- 营业额
- 订单数
- 售币数（含电子币和实物币）——仅收银台连接了售币机时显示
- 币余额数——仅收银台连接了售币机时显示

**离线设备**
- 只显示设备名称 / 设备编号 / 离线状态，不显示营业数据

---

## 四、自助机

**数据范围：** 当日

**预警状态规则**
- 打开看板时读取一次，点击刷新按钮后更新
- 多个预警同时存在时叠加显示所有预警图标
- 预警类型及适用范围：

| 预警类型 | 适用设备 | 判断来源 |
|---|---|---|
| 缺打印纸 | 所有自助机 | 硬件传感器上报 |
| 通讯状态异常 | 所有自助机 | 系统通讯状态 |
| 存币桶满 | 存币机 | 硬件传感器上报 |
| 存票桶满 | 存票机 | 硬件传感器上报 |
| 存珠桶满 | 售珠提珠机 | 硬件传感器上报 |

**各子类型显示字段**

售币提币机
- 营业额
- 售币数（含电子币和实物币）
- 币余额数
- 卡余额

存币机
- 存币数

存票机
- 存票数

售珠提珠机
- 营业额
- 售珠数（含电子弹珠和实物弹珠）
- 存珠数
- 珠余额

手持机
- 营业额

取卡机
- 出卡数
- 卡余额

抽奖机
- 抽奖次数
- 出奖次数

娃娃回收机
- 收娃娃数

快投王
- 投实物币总数
- 投电子币总数
- 投币总数（投实物币总数 + 投电子币总数）
- 三个指标均统计投币正常的数据，含前台投币异常处理中重新投币的数据

**离线设备**
- 只显示设备名称 / 设备编号 / 离线状态，不显示营业数据

---

## 五、游戏机

**数据范围：** 当日

### 5.1 总看板（始终固定显示在顶部）

- 所有机台投币数汇总
- 出票总数
- 出点总数
- 出礼品总数
- 未处理报警数量（点击跳转门店报警列表）

### 5.2 子视图切换

**按类型（默认）**
- 分为：模拟机 / 彩票机 / 中性机 / 礼品机
- 每类下列出每台游戏机明细

**按基站**
- 列出每个基站
- 每个基站下以每台游戏机为单位显示数据

**按机种**
- 以机种为单位显示数据
- 点击机种，弹窗显示列表
- 列表以每个P位为单位显示数据
- 列表包含该机种下每台游戏机的每个P位数据

### 5.3 游戏机列表排序规则

- 默认按净投币数降序排列
- 净投币数为0的排在有数据的后面，再按编号升序排列
- 可切换排序：出彩票数 / 出礼品数量 / 出点
- 出彩票排序：仅对彩票机生效
- 出礼品排序：仅对礼品机生效
- 出点排序：对彩票机、中性机均生效

### 5.4 各类型游戏机显示字段

**模拟机（每台）**
- 投币数

**彩票机（每台）**
- 投币数
- 出票数
- 出点数
- 每币奖票数（出票数 / 投币数）

**中性机（每台）**
- 投币数
- 出点数
- 差额（投币数 - 出点数）
- 盈币率（差额 / 投币数）

**礼品机（每台）**
- 投币数
- 出礼品数
- 出礼品局数（出礼品币数 / 游戏机每局投币数）
- 返还率（出礼品价值 / 投币价值）
- 所选时间内无售币记录时，销售币单价无法计算，返还率显示"--"

**销售币单价计算公式**
- 销售币单价 = （售币套餐实际支付金额 - 退单金额）/（售币数量 - 退单币数）
- 取系统已有的销售币单价计算逻辑

### 5.5 游戏机交互规则

**报警图标**
- 有未处理报警的游戏机显示报警图标
- 点击报警图标跳转该台游戏机报警明细

**点击字段跳转报表**
- 点击投币数 / 出彩票 / 出礼品 / 出点，跳转系统已有对应报表
- 自动带入当前看板时间范围和设备编号作为筛选条件

**P位弹窗**
- 点击任意一台游戏机，弹窗显示各P位数据
- 单P位游戏机也支持点击，显示弹窗
- 弹窗字段与该游戏机卡片当前保留指标一致

**离线设备**
- 只显示设备名称 / 设备编号 / 离线状态，不显示营业数据

---

## 六、大项目

**数据范围：** 按时间筛选，默认今日

**每个大项目显示字段**
- 在场人数（所选时间不含今日时显示"--"）
- 入场人数
- 用门票数（入闸扣的次票 + 期限票入场次数，期限票同一天多次入闸只计1次）
- 投币数（入闸扣的代币和期限币总数 + 超时离场后扣的代币和期限币数）
- 加时金额（会员/散客购买的加时套餐金额）

**点击大项目**
- 跳转系统已有出入场明细报表
- 自动带入当前看板时间范围和项目编号作为筛选条件

**离线设备**
- 只显示设备名称 / 设备编号 / 离线状态，不显示营业数据

---

## 七、异常处理

| 场景 | 处理方式 |
|---|---|
| 设备离线（7天内） | 显示基本信息 + 离线状态，不显示营业数据 |
| 设备离线超7天 | 移出主列表，归入"离线设备"页签 |
| 自助机预警触发 | 叠加显示对应预警图标 |
| 游戏机有未处理报警 | 显示报警图标，点击跳转报警明细 |
| 净投币数为0 | 排在有数据设备后面，按编号升序补位 |
| 所选时间内无售币记录 | 销售币单价无法计算，返还率显示"--" |
| 大项目所选时间不含今日 | 在场人数显示"--" |
| 刷新间隔不足1分钟 | 刷新按钮可点击但不产生效果 |

---

## 八、看板取数说明

### 8.1 收银台

| 指标 | 取数说明 |
|---|---|
| 营业额 | 当天的营业额，不含退单 |
| 订单数 | 当天的订单总数，不含退单 |
| 售币数 | 包含代币、实物币、赠币 |
| 币余额 | 剩余实物币数量 |

### 8.2 自助机

| 指标 | 取数说明 |
|---|---|
| 营业额 | 当天的营业额，不含退单 |
| 订单数 | 当天的订单总数，不含退单 |
| 售币数 | 包含代币、实物币、赠币 |
| 币余额 | 剩余实币数量 |
| 存币数 | 存实物币数量 |
| 存票数 | 存实物票数量 |
| 售珠数 | 包含实物弹珠和存入会员账号的弹珠 |
| 存珠数 | 存实物弹珠数量 |
| 珠余额 | 剩余实物弹珠数量 |
| 出卡数 | 出会员卡数量 |
| 卡余额 | 剩余实物卡数量 |
| 抽奖次数 | 抽奖总次数，包含抽奖成功、失效次数 |
| 出奖次数 | 出奖总次数，已剔除不中奖次数 |
| 投币总数 | 等于投实物币+投电子币 |

### 8.3 游戏机

| 指标 | 取数说明 |
|---|---|
| 投币数 | 包含代币、实物币、赠币、游戏时间投币数、游戏机局数投币数 |
| 出点数 | 包含实物和电子 |
| 净投币数 | 等于投币数-出点数 |
| 出票数 | 出彩票数量，包含实物和电子 |
| 出蓝票数 | 出特殊奖票数，包含实物和电子 |
| 每币奖票数 | 等于出奖数/净投币数(投币数-出点数) |
| 差额 | 等于投币数-出点数-出特殊奖票数 |
| 盈币率 | 等于差额/投币总数*100% |
| 出礼品数 | 等于礼品机掉落数量+礼品机盘存出库数，盘存前一天数据，需隔天才会更新数据；当天盘存则实时运算 |
| 出礼品币数 | 等于投币数/出礼品数 |
| 出礼品局数 | 等于出礼品币数/每局投币数 |
| 投币价值 | 等于投币数*币单价 |
| 出礼品价值 | 出礼品数*礼品成本单价；礼品成本核算后同步更新本期礼品成本单价及出礼品价值 |
| 返还率 | 等于出礼品价值/投币价值 |

### 8.4 大项目

| 指标 | 取数说明 |
|---|---|
| 在场人数 | 当前未离场的人数 |
| 入场人数 | 等于投币+扣次票+通行证当天首次入场人数 |
| 用门票数 | 等于扣次票+通行证 |
| 加时金额 | 会员/散客购买的加时套餐金额 |

---

## 九、原型补充规则

- 设备类型页签右侧显示"看板取数说明"入口。
- 点击"看板取数说明"后弹窗展示收银台、自助机、游戏机、大项目的指标取数口径。
- 游戏机页签顶部汇总指标显示为：投币总数、出票总数、出点总数、出礼品总数、未处理警报。
- 游戏机按机种视图使用与按类型视图一致的游戏机卡片样式。
- 游戏机按机种视图按机种汇总对应类型保留指标。
- 游戏机按机种视图中，礼品机机种卡片不显示出礼品局数。
- 彩票机机种卡片直接显示投币数、出票数、出点数、每币奖票数，不再切换出票数据、出点数据。
- 礼品机机种卡片直接显示投币数、出礼品数、返还率，不再切换出礼品数、返还率。
- 礼品机卡片显示出礼品局数，点击该指标打开游戏机盈收分析。
- P位命名格式为1P、2P、3P。
- 单台游戏机P位弹窗标题只显示游戏机名称。
- 机种P位弹窗标题只显示机种名称。

## 十、后台菜单与报表下钻规则

- 后台菜单按系统主菜单展示：营业、套餐、商品、营销、会员、报表、设备、小程序、运动、设置。
- 当前一级菜单为设备。
- 设备看板作为独立二级菜单显示在扣费设备分组上方。
- 扣费设备分组内不重复显示设备看板。
- 打开设备看板时，右侧工作区按系统标签页方式打开设备看板标签。
- 设备看板二级菜单不显示图标。
- 设备看板卡片不再通过整张卡片打开报表。
- 在线设备通过设备名称打开对应报表。
- 取卡机设备名称不打开报表。
- 游戏机通过指定指标打开游戏机盈收分析。
- 游戏机设备名称保留打开P位明细弹窗。
- 非游戏设备只通过设备名称打开对应报表，卡片指标数值不打开报表。
- 所有报表自动带入当前看板时间和当前设备作为查询条件。
- 所有报表隐藏合作商查询条件。

| 设备/指标 | 打开报表 | 隐藏查询条件 |
|---|---|---|
| 收银台、售币提币机、售珠提珠机、手持机设备名称 | 销售查询 | 开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端、合作商 |
| 存币机设备名称 | 会员存币查询 | 开始时间、结束时间、开始营业日期、结束营业日期、营业班次、设备、合作商 |
| 存票机设备名称 | 存实物票查询 | 开始时间、结束时间、设备、合作商 |
| 取卡机设备名称 | 不打开报表 | 无 |
| 抽奖机设备名称 | 抽奖机记录 | 开始时间、结束时间、合作商 |
| 娃娃回收机设备名称 | 礼品回收记录 | 起始回收时间、结束回收时间、起始营业日期、结束营业日期、操作终端、合作商 |
| 快投王设备名称 | 快投王记录 | 时间、设备、合作商 |
| 游戏机指标：投币数、出票数、出点数、每币奖票数、出礼品数、出礼品局数、返还率、差额、盈币率 | 游戏机盈收分析 | 所有查询条件 |
| 大项目设备名称 | 出入场查询 | 开始入场时间、结束入场时间、开始离场时间、结束离场时间、项目、开始营业日期、结束营业日期、班次、合作商 |

## 十一、本轮补充规则

- 报表弹窗只隐藏需求指定的查询条件。
- 未被隐藏的查询条件必须显示在报表表格上方。
- 报表弹窗查询条件区下方显示查询、导出按钮。
- 报表弹窗不显示“已自动带入当前看板时间和当前设备”的提示文案。
- 顶部搜索栏左侧不显示功能路径“设备/扣费设备/设备看板”。
- 游戏机页签顶部设备看板的投币总数、出票总数、出点总数、出礼品总数均打开游戏机盈收分析。
- 顶部设备看板打开的游戏机盈收分析默认选中按游戏机汇总页签。
- 顶部设备看板打开的游戏机盈收分析只显示按游戏机汇总页签，其余页签隐藏。
- 顶部设备看板打开的游戏机盈收分析隐藏开始营业日期、结束营业日期、合作商。
- 顶部设备看板打开的游戏机盈收分析显示其它查询条件。
- 顶部设备看板打开的游戏机盈收分析中，游戏机编号、游戏机名称列固定，不随横向滚动条拖动。
- 未处理警报打开消息中心。
- 未处理警报数量只统计游戏机未处理异常消息。
- 未处理警报弹窗只显示游戏机警报。
- 库存消息不纳入未处理警报数量统计。
- 未处理警报消息中心不显示库存、自助机标签页。
- 库存消息不显示在未处理警报列表中。
- 消息标记已读后从列表移除，并同步更新未处理警报数量。
- 离线设备显示已离线时间。
- 一个月内按天显示已离线时间。
- 超过一个月按月显示已离线时间。
- 超过一年按年显示已离线时间。
- 删除或失效设备不显示在离线设备和其它所有标签页中。
- 游戏机单机卡片显示P位数量，格式为“P位数:2”。
- 机种卡片不显示P位数量。
- 游戏机页签不显示时间选择，只显示当天数据。
- 游戏机卡片只保留指定指标：模拟机显示投币数；彩票机显示投币数、出票数、出点数、每币奖票数；礼品机显示投币数、出礼品数、出礼品局数、返还率；中性机显示投币数、出点数、差额、盈币率。
- 游戏机按机种视图中，礼品机机种卡片不显示出礼品局数。
