设备看板需求文档

==============================
【一、入口与范围】
==============================

功能名称：设备看板
功能入口：设备模块下独立二级菜单，显示在【扣费设备】分组上方。
展示范围：当前门店有效设备。
不展示范围：删除设备、失效设备。
数据一致性：仅当前门店视角；多门店独立部署，跨门店数据不要求实时一致。
默认页签：全部。

==============================
【二、全局页面规则】
==============================

【后台壳】
1. 顶部系统栏显示：世软科技、切换系统、搜索、审批、下载、知识库、更多、版本、用户信息。
2. 一级菜单【设备】高亮。
3. 右侧工作区以系统标签页打开【设备看板】。

【二级菜单】
1. 二级菜单上方只显示【世软科技】品牌。
2. 二级菜单上方不显示切换系统按钮。
3. 【设备看板】作为独立二级菜单显示在【扣费设备】分组上方。
4. 【扣费设备】分组内不重复显示【设备看板】。

【设备类型页签】
1. 页签顺序：全部、收银台、自助机、游戏机、大项目、离线设备。
2. 默认选中：全部。
3. 离线设备页签显示红点。

【页面滚动】
1. 页面标题、刷新区、设备类型页签、看板取数说明按钮固定在右侧内容区顶部。
2. 垂直滚动时，只滚动下方设备列表。

【刷新】
1. 打开页面自动刷新一次。
2. 点击【刷新数据】刷新看板数据。
3. 距上次刷新不足 1 分钟时，按钮可点击但不更新数据。

【时间筛选】
1. 仅大项目页签显示时间筛选。
2. 大项目默认选中【今日】。
3. 可选项：今日、昨日、本周、本月、自定义。
4. 全部、收银台、自助机、游戏机页签不显示时间筛选。

【固定文案】
1. 所有界面不显示【当日实时数据】。
2. 全部页签中，设备类型标题不显示【区块】。

==============================
【三、设备加载与排序】
==============================

【主列表范围】
1. 显示有效状态设备。
2. 有效状态包含在线设备和 7 天内离线设备。

【离线规则】
1. 离线超 7 天的设备不在主列表显示。
2. 离线超 7 天的设备归入【离线设备】页签。
3. 离线判断基准为设备最后一次上报数据时间。

【全部页签展示顺序】
1. 收银台
2. 自助机
3. 游戏机
4. 大项目

【同类设备排序】
1. 在线设备优先。
2. 同状态内按设备编号升序。

【离线设备显示】
1. 显示设备名称、设备编号、离线状态、已离线时间。
2. 不显示营业数据。
3. 一个月内按天显示已离线时间。
4. 超过一个月按月显示。
5. 超过一年按年显示。
6. 离线设备分组名称显示：收银台、自助机、游戏机、大项目。
7. 不显示【断连】分组名称。

==============================
【四、设备字段】
==============================

【1. 收银台】
数据范围：当日。
在线设备字段：营业额、订单数。
连接售币机时追加字段：售币数、币余额数。
离线设备字段：设备名称、设备编号、离线状态、已离线时间。
离线设备不显示营业数据。

【2. 自助机】
数据范围：当日。
售币提币机字段：营业额、售币数、币余额数、卡余额。
存币机字段：存币数。
存票机字段：存票数。
售珠提珠机字段：营业额、售珠数、存珠数、珠余额。
手持机字段：营业额。
取卡机字段：出卡数、卡余额。
抽奖机字段：抽奖次数、出奖次数。
娃娃回收机字段：收娃娃数。
快投王字段：投实物币总数、投电子币总数、投币总数。

【自助机预警】
缺打印纸：适用所有自助机，来源为硬件传感器上报。
通讯状态异常：适用所有自助机，来源为系统通讯状态。
存币桶满：适用存币机，来源为硬件传感器上报。
存票桶满：适用存票机，来源为硬件传感器上报。
存珠桶满：适用售珠提珠机，来源为硬件传感器上报。

【预警规则】
1. 打开看板时读取一次预警。
2. 点击刷新后更新预警。
3. 多个预警同时存在时叠加显示全部预警图标。

【快投王统计规则】
1. 投实物币总数、投电子币总数、投币总数只统计投币正常数据。
2. 包含前台投币异常处理中重新投币的数据。

【3. 游戏机】
数据范围：当日。
子视图：按类型、按基站、按机种。
默认子视图：按类型。
按类型分组：模拟机、彩票机、中性机、礼品机。
按基站分组：每个基站下显示对应游戏机。
按机种分组：按机种汇总，点击机种打开 P 位列表。

【游戏机排序】
1. 默认按净投币数降序。
2. 净投币数为 0 的设备排在有数据设备后面。
3. 净投币数相同时按设备编号升序。
4. 出彩票数排序仅对彩票机生效。
5. 出礼品数量排序仅对礼品机生效。
6. 出点排序对彩票机、中性机生效。

【P 位规则】
1. 单机卡片显示 P 位数量，格式：P位数:2。
2. 机种卡片不显示 P 位数量。
3. P 位命名格式：1P、2P、3P。

【游戏机字段】
模拟机单机卡片：投币数。
模拟机机种卡片：投币数。
彩票机单机卡片：投币数、出票数、出点数、每币奖票数。
彩票机机种卡片：投币数、出票数、出点数、每币奖票数。
中性机单机卡片：投币数、出点数、差额、盈币率。
中性机机种卡片：投币数、出点数、差额、盈币率。
礼品机单机卡片：投币数、出礼品数、出礼品局数、返还率。
礼品机机种卡片：投币数、出礼品数、返还率。

【游戏机总看板】
1. 指标：投币总数、出票总数、出点总数、出礼品总数、未处理警报。
2. 投币总数、出票总数、出点总数、出礼品总数点击后打开游戏机盈收分析。
3. 未处理警报只统计游戏机未处理异常。
4. 未处理警报点击后打开消息中心。

【游戏机验收点】
1. 游戏机页签按类型视图不显示顶部“游戏机 X 在线 / X 台”总数量条。
2. 模拟机、彩票机、中性机、礼品机分类标题显示对应机台数量。
3. 游戏机按类型分类标题不显示英文状态标签。
4. 彩票机、礼品机卡片尺寸一致。
5. 彩票机、礼品机切换数据视图时卡片高度不变化，指标不遮挡。
6. 彩票机出点数据中的盈币率必须完整显示。

【4. 大项目】
数据范围：按时间筛选，默认今日。
字段：在场人数、入场人数、用门票数、投币数、加时金额。
在场人数规则：所选时间不含今日时显示 --。

==============================
【五、交互与报表下钻】
==============================

【通用报表规则】
1. 所有报表自动带入当前看板时间和当前设备。
2. 报表弹窗只隐藏需求指定的查询条件。
3. 未指定隐藏的查询条件必须显示。
4. 查询条件区下方显示【查询】【导出】按钮。
5. 报表弹窗不显示“已自动带入当前看板时间和当前设备”。
6. 报表弹窗不显示“点击指标”。
7. 除本节明确可点击的设备名称、游戏机指定指标、游戏机总看板指标外，其它指标不打开报表。

【非游戏设备下钻】
收银台设备名称 → 销售查询 → 隐藏开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端、合作商。
售币提币机设备名称 → 销售查询 → 隐藏开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端、合作商。
售珠提珠机设备名称 → 销售查询 → 隐藏开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端、合作商。
手持机设备名称 → 销售查询 → 隐藏开始时间、结束时间、开始营业日期、结束营业日期、营业班次、操作终端、合作商。
存币机设备名称 → 会员存币查询 → 隐藏开始时间、结束时间、开始营业日期、结束营业日期、营业班次、设备、合作商。
存票机设备名称 → 存实物票查询 → 隐藏开始时间、结束时间、设备、合作商。
取卡机设备名称 → 不打开报表。
抽奖机设备名称 → 抽奖机记录 → 隐藏开始时间、结束时间、合作商。
娃娃回收机设备名称 → 礼品回收记录 → 隐藏起始回收时间、结束回收时间、起始营业日期、结束营业日期、操作终端、合作商。
快投王设备名称 → 快投王记录 → 隐藏时间、设备、合作商。
大项目设备名称 → 出入场查询 → 隐藏开始入场时间、结束入场时间、开始离场时间、结束离场时间、项目、开始营业日期、结束营业日期、班次、合作商。

【游戏机下钻】
游戏机设备名称 → 打开 P 位明细弹窗 → 单 P 位游戏机也打开，标题只显示游戏机名称。
游戏机机种 → 打开 P 位明细弹窗 → 标题只显示机种名称，列表以每个 P 位为单位。
游戏机报警图标 → 打开该游戏机报警明细 → 不触发 P 位弹窗。
游戏机指定指标 → 打开游戏机盈收分析 → 隐藏所有查询条件。
游戏机指定指标包括：投币数、出票数、出点数、每币奖票数、出礼品数、出礼品局数、返还率、差额、盈币率。

【游戏机总看板下钻】
1. 投币总数、出票总数、出点总数、出礼品总数 → 打开游戏机盈收分析。
2. 默认选中按游戏机汇总页签。
3. 只显示按游戏机汇总页签，其余页签隐藏。
4. 隐藏开始营业日期、结束营业日期、合作商。
5. 其它查询条件显示。
6. 游戏机编号、游戏机名称列固定，不随横向滚动条移动。

【未处理警报】
1. 未处理警报 → 打开消息中心。
2. 只显示游戏机警报。
3. 不显示库存、自助机标签页。
4. 库存消息不纳入未处理警报数量。
5. 库存消息不显示在未处理警报列表。
6. 消息标记已读后从列表移除，并同步更新未处理警报数量。

【看板取数说明】
1. 点击【看板取数说明】 → 打开取数说明弹窗。
2. 弹窗展示收银台、自助机、游戏机、大项目指标口径。

==============================
【六、异常与验收】
==============================

【设备离线 7 天内】
系统处理：主列表保留，仅显示基础信息和离线状态。
测试验收：不显示营业数据字段。

【设备离线超 7 天】
系统处理：移出主列表，进入离线设备页签。
测试验收：离线页签按收银台、自助机、游戏机、大项目分组。

【删除或失效设备】
系统处理：所有页签均不显示。
测试验收：主列表、离线页签均查不到。

【无营业数据】
系统处理：指标显示 -- 或按设备规则隐藏。
测试验收：页面不报错，不显示误导性 0。

【刷新间隔不足 1 分钟】
系统处理：点击刷新不更新数据。
测试验收：上次刷新时间不变。

【自助机多预警】
系统处理：叠加显示全部预警图标。
测试验收：预警不互相覆盖。

【游戏机未处理报警】
系统处理：显示报警图标，计入未处理警报。
测试验收：点击后只显示游戏机警报。

【所选时间内无售币记录】
系统处理：销售币单价无法计算，返还率显示 --。
测试验收：不显示错误数值。

【大项目非今日】
系统处理：在场人数显示 --。
测试验收：昨日、本周、本月、自定义均验证。

==============================
【七、取数口径】
==============================

【通用】
营业额：当天营业额，不含退单。
订单数：当天订单总数，不含退单。
售币数：代币、实物币、赠币合计。
币余额：剩余实物币数量。

【自助机】
存币数：存实物币数量。
存票数：存实物票数量。
售珠数：实物弹珠和存入会员账号的弹珠合计。
存珠数：存实物弹珠数量。
珠余额：剩余实物弹珠数量。
出卡数：出会员卡数量。
卡余额：剩余实物卡数量。
抽奖次数：抽奖总次数，含成功、失效次数。
出奖次数：出奖总次数，剔除不中奖次数。
投币总数：投实物币总数 + 投电子币总数。

【游戏机】
投币数：代币、实物币、赠币、游戏时间投币数、游戏机局数投币数合计。
出点数：实物出点和电子出点合计。
净投币数：投币数 - 出点数。
出票数：实物彩票和电子彩票合计。
出蓝票数：实物特殊奖票和电子特殊奖票合计。
每币奖票数：出奖数 / 净投币数。
差额：投币数 - 出点数 - 出特殊奖票数。
盈币率：差额 / 投币总数 * 100%。
销售币单价：（售币套餐实际支付金额 - 退单金额）/（售币数量 - 退单币数），取系统已有逻辑。
出礼品数：礼品机掉落数量 + 礼品机盘存出库数；盘存前一天数据隔天更新，当天盘存实时运算。
出礼品币数：投币数 / 出礼品数。
出礼品局数：出礼品币数 / 每局投币数。
投币价值：投币数 * 销售币单价。
出礼品价值：出礼品数 * 礼品成本单价；礼品成本核算后同步更新本期礼品成本单价及出礼品价值。
返还率：出礼品价值 / 投币价值。

【大项目】
在场人数：当前未离场人数。
入场人数：投币入场 + 扣次票入场 + 通行证当天首次入场人数。
用门票数：扣次票 + 通行证。
投币数：入闸扣的代币和期限币总数 + 超时离场后扣的代币和期限币数。
加时金额：会员、散客购买加时套餐金额。
